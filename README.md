# Python-Qrcode-Generator
Qrcode Generator is a free and open source generator designed to help users convert and text into an Qrcode. 

Before running the app for the first time, you need to install the required dependencies:
```
pip install qrcode ttkbootstrap Pillow
```

You also need Subdora installed
```
pip install subdora
```
### Features
* **Modern GUI:** Built using `ttkbootstrap`.
* **Save Option:** Easily export your generated QR codes as image files using a built-in file dialog.

### How to Use
1. Run the app using the Subdora in the command line.
2. Enter the text or URL you want to convert into a QR code.
3. Click the generate button to view your QR code.
4. Use the save button to export the image to your computer.

## How to run Qrcode Generator
```
subdora --run Qrcode_generator.myst
```
